Detailed Description

High-level API class, used for the convenient set-up of low-latency, app-to-app data connections.

The connection follows a datagram model (i.e. there is no guaranteed delivery), as it is based upon UDP. See jibe.sdk.client.simple for code examples.

void DatagramSocketConnection.start ( string  receiverUserId,
string  inviteMessage 
)

Starts a datagram socket to a remote user.

Returns immediately. A SimpleConnectionStateListener has to be used, in order to detect when the connection has been started, or if it fails.

Parameters
receiverUserId- The phone number/SIP URI of user you are trying to connect (e.g. +491235678 or (408) 345-6789)
inviteMessage- A message that can be sent with the request to establish a data connection, for example, "Fancy a game?". This message can be displayed on the remote user's device.
