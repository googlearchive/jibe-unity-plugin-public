void SimpleApiStateListener.onInitializationFailed ( string  message)

The SimpleConnection has established a connection to the Jibe Realtime Engine and is ready to be used.

Creating a SimpleConnection object will trigger the establishment of the service connection (service = Realtime Engine) and consequently a callback to this method.

void SimpleApiStateListener.onInitialized ( string  message)

Initializing the simple connection has failed.

This can happen if the user did not or could not authenticate to the Jibe cloud or if the application is not authorized.
